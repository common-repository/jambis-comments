== Description ==

Jambis is a web service that you easily add the most advanced commenting features to your WordPress blog. You just need to upload one file to your WordPress plugins directory, then activate the plugin. Then all your blog posts will have integrated Jambis comments which included threaded comments, avatars, comment rating, spam control and more. 

== Installation ==

Extract the Zip file to the plugins directory and activate.

You may want to disable the regular WordPress comments as well.
